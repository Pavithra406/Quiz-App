🧠 Quiz Web Application

A simple and interactive Quiz Web Application built using Python (Flask) and basic web technologies.
This project allows users to attempt quiz questions and view their results instantly after completing the quiz.

The application demonstrates the integration of backend logic with a frontend interface, helping in understanding how full-stack web applications work.

🚀 Project Objective

The goal of this project is to:

Learn the fundamentals of web application development
Understand backend development using Python and Flask
Design simple interactive web interfaces
Practice handling user input and responses
Build a basic full-stack application
📂 Project Structure
quiz_site/
│
├── app.py
│
├── templates/
│   ├── HTML files for quiz interface
│
├── static/
│   ├── CSS and JavaScript files
│
└── __pycache__/
🔹 app.py

The main backend file that runs the Flask application and handles the quiz logic such as:

Loading questions
Processing user answers
Calculating the final score
🔹 templates

Contains the HTML pages used to display the quiz interface and results.

🔹 static

Stores CSS and JavaScript files used for styling and client-side interactions.

🔹 pycache

Automatically generated Python cache files.

✨ Features

✅ Interactive quiz interface
✅ Multiple choice questions
✅ Automatic answer evaluation
✅ Score calculation and result display
✅ Simple and user-friendly UI
