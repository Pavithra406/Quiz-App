Quiz Web Application

This repository contains a simple Quiz Web Application developed to demonstrate the fundamentals of web application development using Python and basic front-end technologies. The application allows users to participate in quizzes and view their results after answering a set of questions.

The project was created as part of my learning process to understand how backend logic interacts with frontend interfaces in a web-based environment.

Objective

The main objectives of this project are:

To understand the basics of web application development
To implement backend logic using Python
To design simple user interfaces using HTML and CSS
To learn how frontend and backend components communicate
To gain experience in building interactive web applications
Project Structure
quiz_site/
│
├── app.py
├── templates/
│   └── HTML files for quiz pages
│
├── static/
│   └── CSS, JavaScript, and other static assets
│
└── __pycache__/
app.py

This is the main backend file of the application. It handles the core logic of the quiz system, such as loading questions, processing user answers, and generating results.

templates

This folder contains the HTML files used to display the quiz interface to the users.

static

This folder stores static resources such as CSS and JavaScript files used for styling and client-side interactions.

pycache

This directory contains compiled Python files automatically generated when the application runs.
